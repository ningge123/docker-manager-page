## docker-manager-page

## 使用

1. 获取源码资源包

   ```
   git clone https://github.com/xiaojun207/docker-manager-page.git
   ```

   

2. 安装依赖，国内推荐使用cnpm或tyarn，国外推荐使用npm或yarn

   ```
   npm install
   ```

   

3. 运行

   ```
   npm run dev 或 npm run start
   ```

   

4. 打包

   ```
   npm run build
   ```
